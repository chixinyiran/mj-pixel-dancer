# 🕺 MJ · 像素舞者 × Web Audio

像素迈克尔·杰克逊随音乐节拍起舞 —— 纯前端实现，无依赖，单页面。

## ✨ 特性

- 🎵 4 首经典曲目：Billie Jean / Beat It / Smooth Criminal / Thriller
- 🕺 像素舞者引擎：太空步、踢腿、爪手、僵尸滑步，舞步与音乐时钟同步
- 🎨 每首独立主题配色 + 招牌动作
- ⏯️ 点击舞台 = 当前歌曲播放/暂停切换
- 🔁 连播 / 全屏 / 录制 / 画质切换
- 📱 移动端适配 + 锁屏媒体控件（MediaSession）
- 🎚️ 切歌丝滑淡入淡出

## 🚀 在线体验

打开 [GitHub Pages 链接](#) 即可游玩（建议用 Chrome 以获得最佳后台播放体验）。

## ⌨️ 快捷键

- `空格` 播放/暂停当前歌曲
- `1-4` 切换四首歌
